# upsertShippingMethodAdjustments - Cart Module Reference

This documentation provides a reference to the `upsertShippingMethodAdjustments` method. This belongs to the Cart Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method upserts shipping method adjustments.

## Example

```ts
const shippingMethodAdjustments = await orderModuleService
  .upsertShippingMethodAdjustments(
    [
      {
        shipping_method_id: "123",
        code: "50OFF",
        amount: 5
      },
      {
        id: "321",
        amount: 5
      }
    ]
  )
```
